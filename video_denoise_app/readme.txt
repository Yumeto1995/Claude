====================================================
 video_denoise.py — 動画ノイズ除去ツール 使い方
====================================================

■ 概要
------
動画から音声を抽出し、別途用意したノイズ音源をもとに
スペクトルゲーティングでノイズを除去したあと、
クリーンな音声を元動画に結合して出力します。


■ 必要環境
----------
- Python 3.9 以上
- ffmpeg（別途インストールが必要）

  macOS:
    brew install ffmpeg

  ※ ffmpeg のパスは自動検出されます。PATH が通っていない場合でも
    /opt/homebrew/bin または /usr/local/bin にあれば動作します。


■ Pythonライブラリのインストール
---------------------------------
  pip3 install -r requirements.txt


■ ディレクトリ構成
------------------
  video_denoise_app/
  ├── input/          ← 入力動画を1つ置く
  ├── noise/          ← 参照ノイズ音源を1つ以上置く
  ├── output/         ← 出力動画が保存される（自動生成）
  ├── video_denoise.py
  ├── requirements.txt
  └── readme.txt


■ 対応ファイル形式
------------------
  input/ noise/ ともに以下の形式に対応:
  動画: .mp4 / .mov / .avi / .mkv
  音声: .wav / .mp3 / .flac / .aac / .m4a

  ※ noise/ には動画ファイルをそのまま置くことができます。
    音声は自動的に抽出して使用します。


■ 使い方
---------
1. input/ フォルダにノイズ除去したい動画を1つ置く

2. noise/ フォルダにノイズ音源ファイルを置く
   （複数ファイル可。動画・音声どちらでも対応。
     エアコン音・機械音など実際のノイズを単体で
     録音・撮影したファイルを使うと効果的です）

3. 以下のコマンドを実行する

   python3 video_denoise.py <出力ファイル名>

   例:
     python3 video_denoise.py result.mp4


■ オプション
------------
  --prop-decrease FLOAT
      ノイズ除去の強さ（0.0〜1.0、デフォルト: 1.0）
      値を小さくすると除去が控えめになり、自然な音に近づく

  --n-fft INT
      FFT窓サイズ（デフォルト: 2048）
      大きいほど低周波ノイズに強くなる

  --hop-length INT
      STFTのホップ長（デフォルト: 512）

  --chunk-seconds FLOAT
      チャンク分割サイズ（デフォルト: 60）
      大容量動画でメモリ不足になる場合は小さくする


■ 使用例
---------
  # 基本的な使い方
  python3 video_denoise.py result.mp4

  # ノイズ除去を控えめにする
  python3 video_denoise.py result.mp4 --prop-decrease 0.7

  # 低周波ノイズ（エアコンの低音など）に強くする
  python3 video_denoise.py result.mp4 --n-fft 4096


■ 注意事項
----------
- input/ に置くファイルは必ず1つだけにしてください
- noise/ には複数ファイルを置けます（全て連結してプロファイルを作成します）
- サンプルレートやチャンネル数が異なるノイズファイルも自動で変換します
- 出力動画は映像をそのままコピーし、音声のみ再エンコードします（AAC 192kbps）
